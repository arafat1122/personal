# Arafat Hossen Portfolio

A polished personal portfolio website designed to showcase professional experience, technical skills, and featured projects in a modern dark-themed layout.

## Overview
This project is a static website built with HTML, CSS, and JavaScript. Portfolio content is stored in data.json, making it easy to update the site without editing the page structure directly.

## Features
- Responsive, modern dark UI
- Dynamic content loading from data.json
- Interactive skill filtering by category
- Professional experience timeline section
- Project showcase cards with call-to-action links
- Resume and profile image integration

## Project Structure
- index.html – Main page structure and content rendering logic
- style.css – Visual design, layout, animations, and responsive behavior
- data.json – Portfolio details such as bio, skills, experience, and projects
- profile.jpeg – Profile image used in the hero section
- your-cv.pdf – Resume file linked from the homepage

## Run Locally
You can open index.html directly in a browser, or run a local server for a more reliable preview:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Customization
To update the portfolio content, edit data.json with your own:
- name and bio
- skills and categories
- experience entries
- project descriptions and links

## Deployment
This project can be published on GitHub Pages, Netlify, Vercel, or any static hosting platform.
